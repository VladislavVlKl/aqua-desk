ТРЕНЕР (роль: trainer)
════════════════════════════════════════════════════════════

ГЛАВНАЯ / СПИСАНИЕ ПТ
─────────────────────
— Поиск клиента по имени (поле с debounce, выпадающий список)
— Выбор филиала
— Выбор типа тренировки:
    • Обычная ПТ (списывает с баланса абонемента)
    • Разовое посещение кат.1 / кат.2 / кат.3 (drop-in)
    • Пробная тренировка
    • В долг (без списания баланса)
    • Запрос на позднюю тренировку (старше 48ч, требует одобрения)
— Выбор количества ПТ (1–5 за один раз, пакетное списание)
— Выбор даты и времени каждой ПТ (лимит: не старше 48 часов)
— Запись на другого тренера (замена) — отправляет запрос тренеру Б
— Предупреждение при нулевом балансе клиента
— Блокировка списания если есть незакрытый конспект (требует написать)
— Запись дежурства: начало, конец, филиал

КЛИЕНТЫ
────────
— Список своих клиентов с сортировкой:
    просроченные/нулевой баланс → истекающие → активные
— Цветовая метка: просроченные (красная), истекающие (жёлтая)
— Категория клиента цветным бейджем: кат.1 зелёный, кат.2 фиолетовый, кат.3 жёлтый
— Поиск по имени
— Добавить нового клиента (ФИО, категория, возраст, абонемент)

ПРОФИЛЬ КЛИЕНТА
───────────────
— ФИО, категория, баланс, возраст, статус абонемента
— История тренировок за все время
— Удалить тренировку (в течение 48ч, если нет конспекта)
— Редактировать тренировку (дата, день, сегодня)
— Купить абонемент / пакет ПТ:
    Дети: 5, 10, 25, 50 ПТ / разные сроки
    Взрослые: 5, 10, 25 ПТ
— Закрыть абонемент
— Добавить цель клиента
— Написать конспект тренировки (дедлайн 48ч после ПТ)
— Передать клиента другому тренеру (запрос на перевод)
— Запросить удаление клиента (требует одобрения старшего)
— Редактировать данные клиента (ФИО, категория, возраст, даты абонемента)

СЕГОДНЯ
────────
— Список тренировок на сегодня из расписания
— Одним кликом: списать ПТ или отметить групповое занятие
— Статус каждого слота: выполнено / ожидает

РАСПИСАНИЕ
───────────
— Недельная сетка личного расписания
— Навигация по неделям (← →)
— Цветовые слоты: ПТ фиолетовый, суша жёлтый, вода/группы синий, дежурство серый
— Добавить слот: ПТ, группа, дежурство
— Выбор повторяющийся (дни недели) или разовый (конкретная дата)
— Удалить слот (навсегда или только эту дату)
— Просмотр мероприятий (соревнования, ремонт, другие события)

ОТЧЁТ (МОЙ ОТЧЁТ)
───────────────────
— Навигация по месяцам
— Сводные карточки: кол-во ПТ, разовые, пробные, дежурство (часы), группы, итого ЗП
— Детализация всех тренировок за месяц
— История поздних запросов (статус: ожидает / одобрено / отклонено)
— Подтвердить оплату долга клиентом
— Запросы замен ПТ: принять / отклонить
— Запросы передачи клиентов: принять / отклонить
— Список групповых занятий (взрослые группы) за месяц
— Экспорт в Excel (все ПТ, дежурства)

СОБЫТИЯ
────────
— Список мероприятий (соревнования, квалификации, ремонт)
— Создать событие

ГРУППЫ
───────
— Список назначенных групп (детские и взрослые)
— Кнопка 🔄 Замена — запрос на замену на конкретное занятие

ДЕТСКАЯ ГРУППА:
— Состав группы: имя, возраст, уровень, цена/мес, статус оплаты
— Отметить посещаемость (сегодня)
— Отметить посещаемость за другую дату
— История занятий: все даты с явкой (X / N), редактировать/удалить
— Добавить ребёнка
— Редактировать ребёнка (имя, возраст, цена)
— Архивировать / удалить ребёнка навсегда
— Отметить оплату / снять оплату
— Добавить заметку по прогрессу
— Изменить уровень (Подготовительный / Обучающий / Совершенствование / Спортивный)

ВЗРОСЛАЯ ГРУППА:
— Список участников
— Отметить занятие (явка — количество человек)
— История занятий по месяцам (навигация ← →)
— Добавить/удалить/архивировать участника
— Добавить занятие вручную за прошедшую дату
— Редактировать занятие / удалить занятие

ПРОФИЛЬ ТРЕНЕРА
───────────────
— Просмотр своих данных (ФИО, роль, филиал)
— Сменить PIN-код
— Первичная установка PIN (при первом входе)

УВЕДОМЛЕНИЯ
────────────
— Колокольчик: внутренние уведомления (напоминания, системные)
— Бейдж 📝: количество незакрытых конспектов

КООРДИНАТОР (роль: admin)
════════════════════════════════════════════════════════════

СВОДКА
───────
— Сводная ЗП всех тренеров за месяц по всем филиалам
— Навигация по месяцам
— Фильтр по филиалу
— Детальный отчёт по каждому тренеру:
    ПТ по категориям, дежурства, пробные, разовые, замены, группы
    Премия / штраф (ввести вручную)
    Утвердить финальную ЗП
— Экспорт Excel: отчёт конкретного тренера
— Список поздних запросов за месяц по всему филиалу
— Список замен ПТ за месяц

АНАЛИТИКА
──────────
— Графики и статистика по клиентам, тренировкам, активности
— Контроль активности тренеров: последняя тренировка, количество за период

КЛИЕНТЫ
────────
— Все клиенты всех тренеров всех филиалов
— Поиск, фильтр по тренеру и по истечению абонемента
— Профиль клиента: полная история, редактирование, передача тренеру
— Статистика: всего клиентов, истёкших абонементов

ПЕРСОНАЛ
────────
— Список всех сотрудников
— Добавить тренера / старшего тренера
— Редактировать: ФИО, роль, филиалы
— Дополнительный доступ тренера к другому филиалу
— Удалить сотрудника

ГРУППЫ
───────
— Список всех типов групп по филиалам с назначенными тренерами
— Навигация по месяцам
— Кнопки: 🔄 Замены, 💰 Ставки, ⬇️ Дет.ГП, 📋 Структура, + Тип
— 📋 Структура: все группы по филиалам → группа → дни/время → тренера с ролями
— Добавить тип группы (детская / взрослая, цена, % тренеру)
— Редактировать / удалить тип группы
— Назначить тренера в группу (тренер, тип, филиал, ставка, роль)
— + 2й тренер: добавить второго тренера к существующей группе
— Открепить тренера от группы
— Отчёт детской группы за месяц:
    Посещаемость каждого ребёнка по занятиям
    Статус оплаты (оплачен / не оплачен, сумма, период абонемента)
    ЗП тренерам: авто-расчёт % × сумма оплат, поле корректировки, утвердить
    Руководитель группы: имя + % отчислений → сумма к выплате за месяц
    Экспорт Excel

КОНТРОЛЬ
─────────
— Активность тренеров: последние тренировки, количество за период
— Незакрытые конспекты по всем тренерам
— Запросы на удаление клиентов: одобрить / отклонить
— Поздние запросы тренеров (>48ч): одобрить / отклонить
— Замены ПТ за месяц: установить ставку замены

РАСПИСАНИЕ (координатора)

— Фильтр по филиалу и типу слота

ОПЕРАЦИОНКА
────────────

— Счета: список счетов к оплате
— Закупки: список с приоритетами и статусами (в очереди / получено)
— Хлор: журнал закупок хлора по датам
— Планы и задачи: добавить задачу, статусы (активна / выполнена / отменена)

ЕЩЁ
────

— Настройки доступов тренеров к другим филиалам

АВТОРИЗАЦИЯ
────────────
— Вход через Telegram (tg_id автоматически)
— PIN-код: установить при первом входе, сменить в профиле
— Блокировка на 30 сек после 5 неверных попыток PIN

БЕЗОПАСНОСТЬ
─────────────
— Каждая роль видит только разрешённые данные (RLS в БД)
— PIN хранится в БД в зашифрованном виде (bcrypt)
— Защита от двойного нажатия (все критичные операции)

— Кнопка ← (назад) на всех вложенных экранах
— Навигация по месяцам (‹ ›) в отчётах и расписании

ТАРИФЫ И БИЗНЕС-ПРАВИЛА (зашиты в config.js)
════════════════════════════════════════════════════════════

Ставки ПТ:
    Кат.1 = 85 000 сум
    Кат.2 = 110 000 сум
    Кат.3 = 135 000 сум

Дежурство:
    14 000 сум / час

Взрослые группы (по явке):
    1–3 чел = 110 000 сум / занятие
    4–6 чел = 120 000 сум / занятие
    7+ чел  = 130 000 сум / занятие

Пакеты абонементов:
    Дети:     5 ПТ / 2 нед  |  10 ПТ / 1 мес  |  25 ПТ / 3 мес  |  50 ПТ / 6 мес
    Взрослые: 5 ПТ / 2 нед  |  10 ПТ / 1 мес  |  25 ПТ / 3 мес

Лимиты:
    Тренировку можно внести не старше 48 часов
    Конспект — дедлайн 48 часов после тренировки
    Разовое посещение (drop-in) для ребёнка: только 1 раз
