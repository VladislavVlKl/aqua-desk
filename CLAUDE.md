# AquaDesk — Паспорт проекта

## Что это

Telegram Web App для управления фитнес-центром с бассейном.
Тренеры и координаторы ведут клиентов, тренировки, группы, дежурства и расчёт зарплаты прямо в Telegram.

## Стек

- **Frontend**: Vanilla JS (ES2020), без фреймворков (~7900 строк в app.js)
- **Backend**: Supabase (PostgreSQL + RLS + RPC)
- **Деплой**: GitHub Pages
- **Вход**: Telegram Web App (`window.Telegram.WebApp.initDataUnsafe.user.id`)

## Ключевые бизнес-правила

### Тарифы (config.js)
```
Кат.1 = 85 000 сум/ПТ
Кат.2 = 110 000 сум/ПТ
Кат.3 = 135 000 сум/ПТ
Дежурство = 14 000 сум/час
Группы взрослые: до 3 чел=110к, до 6=120к, 7+=130к
Группы дети: 40% от оплат клиентов (group_children_pct)
```

### Пакеты абонементов (SUB_PACKAGES)
```
Дети:     5 ПТ/2 нед | 10 ПТ/1 мес | 25 ПТ/3 мес | 50 ПТ/6 мес
Взрослые: 5 ПТ/2 нед | 10 ПТ/1 мес | 25 ПТ/3 мес
```

### Разовые посещения
Три варианта: dropin1 (кат.1), dropin2 (кат.2), dropin3 (кат.3).
Дети могут использовать разовое только 1 раз.

### Лимиты дат
- Тренировку можно внести за последние **48 часов** (MAX_BACKDATE_HOURS)
- Конспект — дедлайн **48 часов** после тренировки (NOTE_DEADLINE_HOURS)
- Редактирование записи — **30 минут** (EDIT_WINDOW_MIN)

### PIN-код
Хранится хешированным. Проверка через RPC:
- `verify_pin(p_tg_id, p_pin)` → boolean
- `change_pin(p_profile_id, p_pin)`
- `claim_profile(p_profile_id, p_tg_id, p_pin)`
- `get_profile_by_tg_id(p_tg_id)` → profile

### Кеш
`cached(key, fn, ttl=300000)` — 5 минут в памяти.
Инвалидировать: `invalidateCache('profiles')`, `invalidateCache('branches')`.

### Защита от двойных нажатий
`once(key, fn)` и `_pending` Set — оборачивают критичные операции.

## Реализованный функционал

### Авторизация
- Вход через Telegram tg_id, PIN хешируется в Supabase
- Регистрация: claim_profile (привязка существующего профиля) или создание нового
- Принудительная установка PIN при первом входе

### Панель тренера (`renderTrainerApp`)
| Вкладка | Функции |
|---------|---------|
| Главная | Активное дежурство, быстрый старт дежурства, счётчики (ПТ за месяц, конспекты), незакрытые конспекты |
| Клиенты | Список своих клиентов (поиск, Levenshtein), карточка клиента, добавление, архив |
| Тренировки | Лог ПТ (выбор клиента, даты, типа, замены), список за неделю, запрос позднего внесения |
| Расписание | Недельный вид, слоты (ПТ/дежурство/группа), добавление/удаление слота, пропуск даты |
| Сегодня | Слоты на сегодня с подтверждением/отменой |
| Дежурство | Старт/стоп (только круглые часы), история, редактирование, удаление |
| Мероприятия | Список событий, запись/отмена |
| Отчёт | ПТ и дежурства за месяц, расчёт ЗП, экспорт Excel |

### Карточка клиента
- Баланс (красный при 0 и ниже), абонемент (пакеты, dropin), история абонементов
- Заморозка (calcFreezeResult, сдвиг sub_end), досрочное закрытие (дети — сгорает, взрослые — сохраняется)
- Цели тренировок (add/delete), отчёт по абонементу (группировка по месяцам)
- Перевод к другому тренеру, архивирование с причиной
- Конспект к тренировке (48ч дедлайн)

### Панель старшего тренера (`renderSeniorApp`)
- Аналитика по своему филиалу, отчёт по ПТ и дежурствам
- Группы: назначение/снятие тренера, управление детальной карточкой группы
- Замены: утверждение замен тренеров (ПТ и группы)
- Выплаты тренерам по группам

### Панель координатора (`renderAdminApp`)
| Вкладка | Функции |
|---------|---------|
| Персонал | Добавление/редактирование/архив тренеров, доступ к филиалам |
| Клиенты | Общий список (поиск, фильтр), просмотр карточек |
| Расписание | Расписание всех тренеров по филиалу |
| Аналитика | ПТ по месяцам и тренерам, статистика |
| Итоги / ЗП | Сводная таблица за месяц, детализация по тренеру, корректировки |
| Филиалы | CRUD филиалов |
| Группы | Полное управление: типы, состав, посещаемость, ЗП |
| Контроль | Audit log, мониторинг сессий, конспекты, поздние запросы, запросы на удаление, уведомления, активность тренеров |
| Тех. часть | Оборудование, поломки, закупки, счета, хлор, планы |

### Группы (детские и взрослые)
- Детские: дети (архив/восстановление), посещаемость, оплата за месяц, заметки прогресса, должники, экспорт ЗП
- Взрослые: клиенты, занятия (лог/редактирование/удаление), посещаемость по дате, явка, выплата
- Общее: расписание группы, замены тренеров, привязка instanceId, второй тренер, процентная/flat ставка, надбавка руководителю

### CEO панель (`renderCeoApp`)
- Дашборд: выручка, ПТ, дежурства, группы по филиалам
- ЗП сводка по всем тренерам
- Группы (просмотр)
- Операционка (tech)
- Планы и задачи

### Вспомогательное
- Excel-экспорт: отчёт тренера, итоги по филиалу, детские группы, ЗП группы (xlsx-js-style, цвета, числа)
- Уведомления: in-app + очередь для координатора
- Пробные тренировки (trial sessions)
- Запросы на удаление клиентов (тренер → старший → координатор)
- Позднее внесение тренировок (запрос + одобрение)
